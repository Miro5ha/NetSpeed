# РУКОВОДСТВО НА РУССКОМ ЯЗЫКЕ (RUSSIAN LANGUAGE GUIDE)
# Чем он полезен?
Этот парсер был создан для теста скорости сети со страницы https://www.speedtest.net, а конкретно для парсинга значений скорости интернета.
Его главной особенностью является то что он имеет свой графический интерфейс, а также его можно свернуть в трей и он дальше будет работать и при выходе из парсера юудет создаваться exel файл с данными о скорости во время его работы.
# Что нужно для пользования?
Во первых установить его с помощью установщика NetSpeed Setup который будет в разделе Releases. После чего вы можете запускать парсер. Все сохранения парсер буудет сохранять в папку "Документы" в папку "NetSpeed" (которую он сам создаст) в Windows.
У парсера есть свой установщик т.к. это программа. Интерфейс ТОЛЬКО НА РУССКОМ ЯЗЫКЕ! 
# ENGLISH LANGUAGE GUIDE (РУКОВОДСТВО НА АНГЛИЙСКОМ ЯЗЫКЕ)
# How is it useful?
This parser was created for the network speed test from https://www.speedtest.net, specifically for parsing internet speed values.
Its main feature is that it has its own graphical interface. It can be minimized to the system tray and will continue to run. When you exit the parser, an Excel file with the speed data will be created.
# What do you need to use it?
First, install it using the NetSpeed ​​Setup installer, which will be in the Releases section. Then you can launch the parser. The parser will save all saved files to the "Documents" folder in the "NetSpeed" folder (which it will create) in Windows.
The parser has its own installer because it is a program. GUI IN RUSSIAN ONLY!
